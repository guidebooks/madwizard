# Install Helm v3

This guide shows how to install the Helm CLI. Helm can be installed either from source, or from pre-built binary releases.

=== "MacOS"
    === "Homebrew"
        ```shell
        brew install helm
        ```
    
    === "curl"
        ```shell
        bash $<(curl -L https://raw.githubusercontent.com/helm/helm/main/scripts/get-helm-3)
        ```

=== "Linux"
    ```shell
    bash $<(curl -L https://raw.githubusercontent.com/helm/helm/main/scripts/get-helm-3)
    ```

=== "Windows"
    ```shell
    bash $<(curl -L https://raw.githubusercontent.com/helm/helm/main/scripts/get-helm-3)
    ```
